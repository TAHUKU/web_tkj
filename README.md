# Website Jurusan TKJ - SMK Darut Taqwa

Website profil jurusan Teknik Komputer dan Jaringan (TKJ) yang interaktif, dibangun menggunakan **Node.js**, **Express.js**, dan **Vanilla JavaScript**. Website ini mencakup informasi jurusan, materi pembelajaran, profil guru, dan dokumentasi kegiatan UJIKOM.

## 🚀 Fitur Utama
- **Sistem Navigasi Modern**: Navigasi responsif dengan penanda halaman aktif.
- **Data Dinamis (API)**: Data guru dan dokumentasi UJIKOM diambil langsung dari Back-End (Server-side)[cite: 11, 12].
- **Animasi On-Scroll**: Efek kartu muncul saat di-scroll untuk pengalaman pengguna yang lebih baik[cite: 12].
- **Stabilitas Layout**: Desain CSS yang mencegah "layout shift" atau getaran saat memuat gambar dokumentasi[cite: 13].

## 🛠️ Tech Stack
- **Back-End**: Node.js & Express.js.
- **Front-End**: HTML5, CSS3 (Modern Flexbox/Grid), & Vanilla JavaScript[cite: 1, 3, 10, 12].
- **Font**: Poppins via Google Fonts[cite: 1].
